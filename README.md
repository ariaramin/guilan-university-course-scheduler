# برنامه‌ریز انتخاب واحد دانشگاه گیلان

Chrome Extension محلی مبتنی بر Manifest V3 برای استخراج دروس ارائه‌شده SADA، مرور سریع فهرست‌های بزرگ، تطبیق با چارت درسی و ساخت خودکار برنامه‌های بدون تداخل است. افزونه ثبت انتخاب واحد انجام نمی‌دهد و داده‌ای به سرور خارجی نمی‌فرستد.

> این ابزار مستقل است و وابستگی رسمی به دانشگاه گیلان ندارد.

## نصب و استفاده

1. در Chrome به `chrome://extensions` بروید و `Developer mode` را روشن کنید.
2. `Load unpacked` را بزنید و پوشه `extension/` را انتخاب کنید.
3. صفحه «گروه‌های درسی ارائه شده» را در SADA باز کنید و آیکن افزونه را بزنید.
4. Dashboard دروس را از iframe هم‌دامنه می‌خواند و پیشنهادها را خودکار می‌سازد؛ فایل واحد اجباری نیست.
5. برای نتیجه دقیق‌تر، در بخش «چارت درسی من» فایل DOCX را رها کنید. فایل قدیمی DOC باید ابتدا در Word به DOCX تبدیل شود.

## قابلیت‌های نسخه 0.9.2

- رابط فارسی RTL با فونت محلی Vazirmatn و Data Grid مجازی؛ فقط ردیف‌های viewport وارد DOM می‌شوند و جدول در عرض کوچک اسکرول افقی کنترل‌شده دارد.
- جست‌وجو، فیلتر ظرفیت/تداخل/روز/واحد/مقطع/ترم/چارت/جنسیت مجاز، مرتب‌سازی و گروه‌بندی جمع‌شونده.
- دریافت زنده با handshake نسخه‌دار، تزریق کنترل‌شده در صفحه مجاز، کنترل به‌روزرسانی دستی، پایش تغییرات همان صفحه و نمایش صریح زمان و وضعیت داده ذخیره‌شده.
- فیلترهای اصلی همیشه در دسترس و bottom sheet موبایل با بازگشت focus و پشتیبانی Escape.
- جزئیات درس در dialog و انتخاب اجباری، ترجیحی یا حذف از پیشنهادها.
- خواندن مستقیم ZIP/XML فایل DOCX، بازسازی `gridSpan`/`vMerge` و جدول‌های کنارهم؛ Mammoth فقط fallback ساختاری است. فایل خام ذخیره یا ارسال نمی‌شود.
- بازبینی، ویرایش، حذف و تأیید صریح ردیف‌ها پیش از اعمال چارت، همراه گزارش فنی بدون نمایش محتوای حساس.
- تطبیق مرحله‌ای عنوان خام، کد، نام نرمال‌شده، alias و fuzzy با آستانه سخت و تأیید انسانی.
- نگه‌داشتن درس‌های بدون واحد در جدول و بخش اصلاح، و حذف آن‌ها از پیشنهادها تا ثبت واحد معتبر.
- رتبه‌بندی پیشنهادهای دارای مجموع قطعی: هدف دقیق، نزدیک‌ترین کمتر و سپس نزدیک‌ترین بیشتر.
- نمایش حداکثر ۸ پیشنهاد در ابتدا، بارگذاری مرحله‌ای و مقایسه دکمه‌ای حداکثر دو برنامه بدون checkbox.
- خروجی CSV و Excel-compatible برای برنامه‌های مقایسه‌شده و PDF چاپی A4 مستقل برای هر پیشنهاد.
- هویت بصری دانشگاه گیلان با همان فایل رسمی ارائه‌شده، بدون برش، بازطراحی یا تغییر رنگ، در سربرگ، حالت‌های خالی، بخش درباره و خروجی PDF نمایش داده می‌شود.
- اعلان استقلال ابزار، skip link، راهنمای پنهان جدول و focus trap صفحه فیلتر موبایل برای دسترس‌پذیری بهتر اضافه شده است.

## هویت بصری

منبع لوگو فایل رسمی با نام `UniversityOfGuilan1-2.jpg` است که توسط صاحب پروژه ارائه شده و نسخه بایت‌به‌بایت یکسان آن در `extension/assets/university-of-guilan-logo.jpg` نگهداری می‌شود. نسبت تصویر ۱۴۶۷×۱۷۵۰ حفظ شده و لوگو همیشه روی سطح سفید با فضای تنفس نمایش داده می‌شود. چون این فایل یک قفل‌نمای عمودی کامل است و دارایی رسمیِ مربع یا emblem-only ارائه نشده، عمداً به آیکن کوچک نوار ابزار Chrome تبدیل نشده است؛ چنین تبدیلی در اندازه ۱۶px ناخوانا و مستلزم برش هویت رسمی بود.

PDF، Excel، CSV و JSON دیگر ورودی چارت نیستند. فرمت قدیمی DOC مستقیماً پردازش نمی‌شود؛ برای جلوگیری از خواندن ناقص باید در Word به DOCX تبدیل شود.

## حریم خصوصی و امنیت

- مجوزهای `storage` و `scripting` و host permission محدود به `https://sada.guilan.ac.ir/*` برای پیام‌رسانی و بازیابی کنترل‌شده content script؛ مجوز عمومی `tabs` وجود ندارد.
- بدون دسترسی به cookie/token، بدون endpoint خارجی و بدون remote code.
- نوع فایل با magic bytes بررسی و حجم چارت به ۲۰ مگابایت محدود می‌شود.
- فقط داده ساخت‌یافته استخراج‌شده و اصلاحات کاربر در `chrome.storage.local` می‌ماند؛ فایل خام نگهداری نمی‌شود.

## توسعه و تست

وابستگی‌های زمان توسعه با `npm install` نصب می‌شوند؛ buildهای مرورگری موردنیاز داخل `extension/vendor/` بسته‌بندی شده‌اند تا افزونه آفلاین باشد.

```sh
npm test
```

جزئیات معماری و قراردادها در [docs/technical-design.md](docs/technical-design.md) است.
