# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Instead, please report security issues privately via GitHub Security Advisories by clicking "Report a vulnerability" under the Security tab of this repository.

⚠️ **CRITICAL:** When reporting a vulnerability:
- DO NOT share passwords.
- DO NOT share Session Tokens or Cookies.
- DO NOT share any personal university account data.

Only provide steps to reproduce, the nature of the vulnerability, and safely redacted logs if necessary.
